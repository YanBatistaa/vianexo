# ViaNexo 0.3.4

> Canal: stable
> Data: 2026-05-09
> Base: commits desde v0.3.3

## Highlights
- Sincronizacao em nuvem via Supabase Auth para restaurar dados apos reinstalacao.
- Novo painel de Configuracoes para conectar, sincronizar e restaurar dados da nuvem.

## Improvements
- Banco remoto compartilhado por empresa com RLS no Supabase.
- Exportacao/restauracao local coberta por teste automatizado de payload de sync.
- Skill local `react-doctor` e configuracao de diagnostico para Electron/Vite.
- Refino de estado no React e reducao de warnings do React Doctor.

## Fixes
- Corrige falsos positivos e pontos de estado obsoleto em formularios e rotas.

## Upgrade Guide
- Instale pelo instalador publicado na release.
- O app aplica migrations Prisma automaticamente ao abrir.
- Faca backup local antes de restaurar dados antigos.
- Configure `SUPABASE_URL` e `SUPABASE_PUBLISHABLE_KEY` no ambiente do app instalado para habilitar a nuvem.
